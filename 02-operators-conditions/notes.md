# Operators & Conditions (JavaScript)

## Operators

### Arithmetic
- + → addition
- - → subtraction
- * → multiplication
- / → division
- % → modulus

### Comparison
- == → equal (loose)
- === → equal (strict)
- != → not equal
- > < >= <= → comparisons

### Logical
- && → AND
- || → OR
- ! → NOT

---

## Conditions

### if
Used to execute code based on condition

### if-else
Executes alternative block

### else if
Multiple conditions

Example:
let age = 18;

if(age >= 18){
  console.log("Adult");
} else {
  console.log("Minor");
}
