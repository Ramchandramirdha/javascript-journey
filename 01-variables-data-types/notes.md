# Variables & Data Types (JavaScript)
- let -> value can be changed
- const -> value cannot be changed
- var -> old way (avoid)

## Data Types
- String -> "Hello"
- Number -> 10, 20.5
- Boolean -> true / false
- Undefined -> declared but not assigned
- Null -> intentionally empty

## typeof
Used to check data type

Example :
... javascript
let name = "Ram";
console.log(typeof name); //string
...
